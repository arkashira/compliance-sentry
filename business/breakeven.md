# breakeven.md

## Unit Economics & Break-even Analysis for Compliance Sentry

### Cost per Active User
- **Compute Costs**: $0.10/user/month  
  (Assuming average usage of 1 vCPU for 720 hours at $0.15/hour)
  
- **Storage Costs**: $0.05/user/month  
  (Assuming 10 GB of storage per user at $0.005/GB/month)

- **Bandwidth Costs**: $0.02/user/month  
  (Assuming 5 GB data transfer per user at $0.004/GB)

**Total Cost per Active User**:  
**$0.10 + $0.05 + $0.02 = $0.17/user/month**

### Pricing Tiers
1. **Basic Tier**: $10/month  
   - Features: Basic consent management, Data protection guidelines, Email support

2. **Pro Tier**: $25/month  
   - Features: All Basic features, Advanced analytics, Customizable templates, Priority email support

3. **Enterprise Tier**: $50/month  
   - Features: All Pro features, Dedicated account manager, Compliance audits, API access

### Customer Acquisition Cost (CAC) Range
- Estimated CAC: $50 - $100  
  (Based on industry benchmarks for SaaS products in the compliance and data protection space)

### Lifetime Value (LTV) Estimate
- Average Customer Lifespan: 24 months  
- Average Revenue per User (ARPU): $25/month (weighted average of pricing tiers)
  
**LTV Calculation**:  
**LTV = ARPU × Average Customer Lifespan**  
**LTV = $25 × 24 = $600**

### Break-even Users Count
- Monthly Fixed Costs: $1,000 (including marketing, salaries, and operational costs)
  
**Break-even Users Calculation**:  
**Break-even Users = Monthly Fixed Costs / (Average Revenue per User - Cost per Active User)**  
**Break-even Users = $1,000 / ($25 - $0.17) = $1,000 / $24.83 ≈ 40 users**

### Path to $10K MRR
- Target Monthly Recurring Revenue (MRR): $10,000

**Required Users Calculation**:  
- Assuming a mix of tiers, we can estimate the number of users needed for each tier to reach $10K MRR.

1. **Basic Tier**:  
   - Price: $10  
   - Users Needed: $10,000 / $10 = 1,000 users

2. **Pro Tier**:  
   - Price: $25  
   - Users Needed: $10,000 / $25 = 400 users

3. **Enterprise Tier**:  
   - Price: $50  
   - Users Needed: $10,000 / $50 = 200 users

### Conclusion
To achieve a sustainable business model, targeting a mix of 200 Enterprise users or 400 Pro users would be ideal, as they provide a balance of revenue and user engagement while minimizing the total user count needed to reach $10K MRR.
